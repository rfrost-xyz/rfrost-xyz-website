## Usage

### Pre-requisites
* [Hugo](https://gohugo.io/getting-started/installing/) (version 0.84.3, or above)

### Serve locally

1. Clone the repo.
2. Initialise sub-modules:
   
	```
	git submodule init
	git submodule update
	```

2. Build the site
	```
	hugo serve
	```
3. View result at [localhost:1313](http://localhost:1313/).
